Déchiffreur de Messages

Bienvenue sur notre déchiffreur de messages ! Cet outil vous permet de déchiffrer des messages cryptés à l'aide d'un algorithme de substitution simple.

Comment ça marche ?

Notre déchiffreur utilise un algorithme de substitution, ce qui signifie qu'il remplace chaque lettre du message crypté par une autre lettre selon une certaine règle.

Dans notre cas, nous utilisons un décalage de trois lettres vers la droite dans l'alphabet. Par exemple, si la lettre "A" est cryptée, elle sera remplacée par la lettre "X", la lettre "B" par la lettre "Y", et ainsi de suite.

Commencez par sélectionner votre fichier a crypté dans l'outil. Vous verrez alors un fichier avec le message crypté se créer nommée "message crypté"

Pour déchiffrer votre message, il vous suffit de cliqué sur déchifrer et choisir le fichier message_crypté, une fois choisit un second fichier message-décrypte apparaitra avec vorte message decrypter.
